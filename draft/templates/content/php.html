PHP

PHP (Hypertext PreProcessor) is a programming language that can be embedded into HTML code. The main purpose of this is to allow server-side scripting to occur. This essentially means that the code is run on the server, and these results are the sent to the user. This means that the user can not ‘see’ the code behind the scenes

< ?php
//PHP goes here
?>

Starting a PHP Program

A PHP script can begin anywhere in the document with the file extension of “.php”.

A simple example of a PHP program that prints out “Hello World” to the screen can be seen below:

< !DOCTYPE html >
< html >
< body >
< ?php
echo "Hello World!"
?>
< /body >
< /html >
