HTML/CSS

HyperText Markup Language, or HTML, is the code and language used to develop the structure and form of a website. This works in conjunction with Cascading Style Sheets, or CSS, which gives this structure style, aesthetic appeal and an eye-catching appearance.

More on HTML

The foundational characteristic of HTML is its use of Markup. This enables encapsulation of certain data to inherit certain aspects of that markup. These are also commonly known as tags, which are constructed using elements.

Elements

These designate the definitions of the structure and the content of objects within the webpage. Common elements include heading allocation elements ( h1, h2, ... , hn elements) and the paragraph element ( p ). This can be identified and established with an opening ‘less than’ sign ( < ), and ending in the ‘greater than’ sign ( > ). Both of these should appear on either side of the element name. An example can be seen below.
< h1 >

Tags

A HTML element usually consists of a start and end tag, encapsulating the content to be affected. The opening tag is created as the element, beginning with a less than sign, elements name, then the greater than sign.

< p >

Then to close this, the same thing is applied. However, there is a forward slash placed before the element name.
< /p >

When this is put together, anything in between the opening and closing tags will have the element name applied to it. For example;
< p > This is a paragraph. < /p >

This example will apply the p, or paragraph, attributes onto the content in between the tags. This simple yet effective means for marking content with a specific tag allows for the organisation and separation of data.

Attributes

Attributes are placed within the opening tag, after the elements name. This adds additional properties to the element. An attribute generally includes an attribute name, and an attribute value. An example of this can be seen below;
< p align=”center” > This paragraph will be centered.< /p >

Building a Website

HTML must be initially set up in a certain way. An example can be seen below.
< !DOCTYPE html >
< html >
    < head >
        < title > Hello World! < /title >
    < /head >
    < body >
        < p > Hello World, again! < /p >
    < /body >
< /html >

  • !DOCTYPE html - Line 1 is a document declaration type, which informs the browser which version of HTML is going to used from thereon.
  • html - Line 2 informs the beginning of the html document
  • head - Establishes special parameters of the website
    • title - Changes the tab name to what you allocate it to be
  • body - Establishes the body of the document
  • p - Specifies the area to be in paragraph form

More on CSS

CSS dictates the appearance of a website. CSS can be implemented in three main ways:

  • External style sheet
  • Internal style sheet
  • Inline style

The easiest way to manage a website is to use an external style sheet. From this single style, it single handley holds all the power to change its looks. To do so, the HTML page must make a direct link to the style sheet to be used. This can be done in a single line, as seen below:
< head >
< link rel="stylesheet" type="text/css" href="style.css" >
< /head >
